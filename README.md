# Compiler

I'm trying to build my own compiler from scratch.
I plan to do everything on plain C.

This project is intended solely for learning purposes; do not expect major innovations.

## Tasks
 - Tokenizer
 - Preprocessor
 - Lexer
 - Machine Code generator
